# Onboarding mocks

This folder is reserved for the onboarding mock screens shared in chat.

Recommended filenames:

1. `01-welcome.png`
2. `02-name.png`
3. `03-gender.png`
4. `04-weight.png`
5. `05-body-fat.png`
6. `06-activity.png`
7. `07-goal.png`
8. `08-summary.png`
9. `08-summary-alt.png`

Screen mapping:

- `01-welcome.png`: landing screen with PSMF intro, benefit bullets, disclaimer, and `Pocni` CTA
- `02-name.png`: name entry screen with warm confirmation card
- `03-gender.png`: gender selection screen with visible category thresholds
- `04-weight.png`: weight screen with kg/lbs toggle and converted value
- `05-body-fat.png`: manual body-fat screen with slider and range card
- `06-activity.png`: activity selection screen with live protein target preview
- `07-goal.png`: goal selection screen with recommended option
- `08-summary.png`: final summary screen with protein target, calorie estimate, and CTA
- `08-summary-alt.png`: alternate or secondary capture of the summary screen

Note:

- The current toolchain can create this folder and manifest, but it cannot save binary image attachments from chat directly to disk.
- The current files in this folder already follow this naming convention.
